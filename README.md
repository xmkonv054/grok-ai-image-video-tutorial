## 怎么用Grok生成图片?

免费用户每两小时能生成10次图片。虽然Grok提供了一些预设提示词,但大多数人还是喜欢自己写描述。操作很简单:

- **打开Grok菜单:** 找到输入框

![Grok聊天框界面](image/11778575.webp)

- **输入描述:** 比如"画一只乌龟和一只狗在秋天的城市里玩耍"。Grok会根据你的描述生成两张图片。
- **查看结果:** 选择你喜欢的那张,或者把两张都分享出去。

![Grok生成图片的工具栏,包括分享、导出和反馈选项](image/6830296499.webp)

### 图片工具说明

生成的图片下面有6个按钮:

1. **发帖链接:** 直接把图片发到X上
2. **复制到剪贴板:** 复制图片链接,可以分享到任何地方
3. **分享:** 通过邮件、短信或其他App分享
4. **重新生成:** 如果不满意,用同样的描述再生成一次
5. **点赞:** 告诉Grok这张图片符合你的要求
6. **点踩:** 告诉Grok这张图片不行

选择**发帖链接**后,Grok会自动创建一条帖子,包含你的图片和生成时用的描述。

![从Grok直接发布生成的图片](image/68348177.webp)

## 怎么编辑Grok生成的图片?

Grok最近加入了图片编辑功能,给你几种修改方式。

最简单的方法是在输入框里选**编辑图片**按钮,然后输入修改要求。但这种方式会同时修改两张图片,效果不一定理想。

![编辑Grok图片的操作指南](image/180303990.webp)

更好的办法是把鼠标放在某张图片上,点击**编辑**按钮。这样可以单独修改这张图片,输入更具体的修改要求。

![单独编辑某张Grok图片](image/6840690032248673.webp)

这种方式更适合调整背景或整体风格。而且Grok会生成两个修改版本,你可以选择更好的那个,或者继续修改。

![下载和继续修改Grok编辑后的图片](image/37587686825.webp)
